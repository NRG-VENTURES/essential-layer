1
Enable the Cart Drawer
Urgent ~ 5 minutes Shopify Admin
▼
What's Wrong
The cart drawer is disabled. When customers click "Add to Cart," they're taken to a separate cart page instead of seeing a slide-out drawer. This adds unnecessary clicks and causes drop-offs.
Step-by-Step Instructions
  1. Log into Shopify Admin → Online Store → Themes
  2. Click "Customize" on the live Fame theme
  3. Open Theme Settings (gear icon, bottom-left)
  4. Navigate to Cart section
  5. Change Cart type from "Page" to "Drawer"
  6. Click Save
Expected Result
A slide-out cart drawer appears when items are added, allowing customers to continue shopping or proceed to checkout without leaving the page.
2
Add H1 Headings to All Key Pages
Urgent ~ 30 minutes Theme Developer
▼
What's Wrong
The homepage and collection pages are missing <h1> heading tags. Google uses H1 as the primary signal for what a page is about. Missing H1 = weaker SEO rankings.
Step-by-Step Instructions
  1. Go to Shopify Admin → Online Store → Themes → Edit Code
  2. Open index.liquid (or the homepage section file)
  3. Add an H1 tag near the top of the page content
  4. Repeat for each collection template in collection.liquid
Example Code
<!-- Homepage --> <h1 class="visually-hidden"> Hypoallergenic Organic Cotton Clothing for Sensitive Skin </h1> <!-- Collection Page --> <h1>{{ collection.title }}</h1>
3
Add Open Graph & Twitter Card Meta Tags
Urgent ~ 1 hour Theme Developer
▼
What's Wrong
When someone shares a Cottonique link on Facebook, Instagram, LinkedIn, or Twitter, it shows as a plain text link with no image preview and no description. This kills social engagement and looks unprofessional.
Step-by-Step Instructions
  1. Go to Edit Code → open theme.liquid
  2. Add the meta tags inside the <head> section
  3. Test with Facebook Sharing Debugger and Twitter Card Validator
Example Code (add to theme.liquid <head>)
<meta property="og:title" content="{{ page_title }}"> <meta property="og:description" content="{{ page_description | escape }}"> <meta property="og:image" content="{{ page_image | img_url: '1200x630' }}"> <meta property="og:url" content="{{ canonical_url }}"> <meta property="og:type" content="website"> <meta name="twitter:card" content="summary_large_image"> <meta name="twitter:title" content="{{ page_title }}"> <meta name="twitter:description" content="{{ page_description | escape }}"> <meta name="twitter:image" content="{{ page_image | img_url: '1200x630' }}">
4
Add Product Schema Markup (JSON-LD)
High ~ 2 hours Theme Developer
▼
What's Wrong
Product pages lack structured data. This means Google can't show star ratings, prices, or stock status in search results. Competitors with rich snippets get significantly higher click-through rates.
Step-by-Step Instructions
  1. Go to Edit Code → find the product template (e.g., product.liquid or main-product.liquid)
  2. Add the JSON-LD script block at the bottom of the template
  3. Test with Google Rich Results Test
Example Code
<script type="application/ld+json"> { "@context": "https://schema.org", "@type": "Product", "name": "{{ product.title | escape }}", "image": "https:{{ product.featured_image | img_url: '1024x' }}", "description": "{{ product.description | strip_html | escape }}", "sku": "{{ product.selected_or_first_available_variant.sku }}", "brand": { "@type": "Brand", "name": "Cottonique" }, "offers": { "@type": "Offer", "price": "{{ product.price | money_without_currency }}", "priceCurrency": "USD", "availability": "https://schema.org/{% if product.available %}InStock{% else %}OutOfStock{% endif %}" } } </script>

Remove Duplicate & Unused Third-Party Scripts
High ~ 2-3 hours Shopify Admin
▼
What's Wrong
The site loads 10+ third-party scripts on every page — including Rebuy Engine loading twice. This significantly slows page load and hurts Google Core Web Vitals scores (which affect SEO ranking).
Scripts Found
  • Judge.me — Reviews (keep)
  • Klaviyo — Email marketing (keep, but defer loading)
  • Rebuy Engine — Upsells (LOADED TWICE — fix immediately)
  • Globo Filters — Collection filters (evaluate if needed)
  • Boost Sales / Avada — Pop-ups (evaluate ROI)
  • hCaptcha — Bot protection (keep, lazy-load)
  • Various tracking pixels — Audit which are active
Step-by-Step Instructions
  1. Go to Shopify Admin → Apps → review all installed apps
  2. Remove the duplicate Rebuy script (check theme code for hardcoded instances)
  3. For each app, decide: Keep / Defer / Remove
  4. Run Google PageSpeed Insights before and after to measure improvement

Add Trust Badges Near "Add to Cart"
High ~ 1-2 hours Theme Developer
▼
What's Wrong
Product pages lack visible trust signals near the purchase button. No payment icons, no security badge, no key selling points. Customers shopping for hypoallergenic products are especially cautious — they need reassurance before buying.
What to Add (Below "Add to Cart" button)
  • Payment method icons (Visa, Mastercard, Amex, PayPal)
  • Security badge ("Secure Checkout" with lock icon)
  • Key differentiators in icon format:
    • "100% Organic Cotton"
    • "Hypoallergenic Certified"
    • "Free Returns"
    • "Ships Worldwide"
How to Implement
  1. In Theme Customize, add an Image with Text or Custom Liquid block below the Add to Cart button in the product template
  2. Create a small icon bar with 4 trust badges (use SVG icons for crisp display)
  3. Alternatively, install a free trust badge app from the Shopify App Store
7
Fix Color Contrast for Accessibility
Medium ~ 1 hour Theme Developer
▼
What's Wrong
Several text colors fail WCAG AA accessibility standards. Gray text (#696969) on white backgrounds has a contrast ratio of only 4.0:1 — the minimum required is 4.5:1. This makes text hard to read for visually impaired users and may create legal risk under ADA compliance.
What to Change
/* BEFORE (fails WCAG AA) */ color: #696969; /* ratio: 4.0:1 — FAILS */ color: #45705f; /* ratio: ~4.2:1 — borderline */ /* AFTER (passes WCAG AA) */ color: #595959; /* ratio: 5.3:1 — PASSES */ color: #3a5e50; /* ratio: 5.1:1 — PASSES */
Step-by-Step Instructions
  1. Go to Edit Code → open theme CSS files
  2. Search for #696969 and replace with #595959
  3. Search for #45705f and replace with #3a5e50
  4. Test with WebAIM Contrast Checker

Write Unique Meta Descriptions for All Pages
Medium ~ 2-3 hours Content / Marketing
▼
What's Wrong
Most collection pages and the About page have missing or generic meta descriptions. Google auto-generates snippets that are often less compelling, resulting in lower click-through rates from search results.
Pages That Need Meta Descriptions
  • Homepage — review and optimize existing
  • Each collection (Women's, Men's, Underwear, Bras, etc.)
  • About Us page
  • Top 10 best-selling products (at minimum)
Examples (150-160 characters each)
Women's Collection: "Shop hypoallergenic women's clothing made from 100% organic cotton. Designed for sensitive skin and allergies. Free shipping on orders over $75." Underwear Collection: "Cottonique organic cotton underwear — latex-free, chemical-free, and dermatologist-tested. Relief for eczema, contact dermatitis, and skin allergies."
Where to Add
  1. For collections: Shopify Admin → Products → Collections → select collection → scroll to "Search engine listing" → Edit
  2. For pages: Shopify Admin → Online Store → Pages → select page → scroll to "Search engine listing" → Edit
9
Add Descriptive Alt Text to All Images
Medium ~ 2-4 hours Content / Marketing
▼
What's Wrong
Product and hero images lack descriptive alt text. This hurts Google Image Search visibility (a free traffic source) and makes the site inaccessible to screen reader users.
Alt Text Formula
Pattern: "Cottonique [Product Name] - [Key Feature] - [Color/Variant]" Examples: "Cottonique Women's Hypoallergenic Bralette - 100% Organic Cotton - Natural White" "Cottonique Men's Crew Neck T-Shirt - Latex-Free Elastic - Melange Grey" "Cottonique Organic Cotton Ankle Socks - Seamless Toe - 3-Pack Natural"
Where to Add
  1. Product images: Shopify Admin → Products → select product → click on each image → "Add alt text"
  2. Hero/banner images: Theme Customize → select the image block → fill in the "Image alt text" field
  3. Start with best-selling products and homepage images first

Expand the About Us Page
Medium ~ 3-4 hours Content / Marketing
▼
What's Wrong
The About page has minimal content. For a niche brand serving people with allergies and sensitive skin, this is a huge missed opportunity. These customers do extensive research before buying — they want to deeply trust your brand.
Sections to Add
  • Founder Story — Why was Cottonique created? What personal experience drove it?
  • Mission Statement — "We exist to..." (clear and emotional)
  • Certifications — Oeko-Tex, GOTS, organic cotton certifications with badge images
  • Manufacturing Process — How are the garments made? What makes them different?
  • Conditions We Help — Eczema, contact dermatitis, MCS, latex allergy, etc.
  • Testimonials — Real customer stories with before/after experiences
  • Press & Medical Endorsements — Any doctor recommendations or media features
Tips
  • Use high-quality photos of the team and manufacturing facility
  • Include a video if possible (even a 60-second founder message)
  • Link to certification verification pages
  • Aim for 800-1200 words minimum
